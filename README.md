# How Spring Works

Spring Frameworkni noldan organish uchun tuzilgan kuchli va tushunarli qollanma.

## Nima uchun bu loyiha?

Spring Framework - Java dunyosidagi eng mashhur frameworklardan biri. Lekin kopchilik uning ichki mexanizmlarini tushunmaydi, faqat ishlatishni biladi. Bu loyiha Springni "qanday ishlatish" emas, balki "qanday ishlaydi" degan savolga javob beradi.

## Loyiha tuzilishi

```
how-spring-works/
 |
 +-- README.md
 +-- CONTRIBUTING.md
 +-- LICENSE
 |
 +-- 01-spring-core/       -> Spring asoslari
 +-- 02-spring-boot/       -> Spring Boot ichki mexanizmlari
 +-- 03-rest-api/          -> REST API lar
 +-- 04-data-jpa/          -> JPA va Hibernate
 +-- 05-spring-security/   -> Xavfsizlik
 +-- 06-database/          -> Malumotlar bazasi
 +-- 07-testing/           -> Testlash
 +-- 08-docker-deployment/ -> Deploy qilish
 +-- examples/             -> Amaliy misollar
```

## Qanday organish kerak?

1. **01-spring-core** dan boshlang - bu eng asos
2. Keyin **02-spring-boot** ga oting
3. Qolgan bolimlarni ozingizga kerakli tartibda organing
4. Har bir bolimdan keyin **examples/** dagi kodlarni korib chiqing

## Talablar

- Java 17+
- Gradle (tavsiya etiladi) yoki Maven
- IntelliJ IDEA (tavsiya etiladi)
- PostgreSQL (malumotlar bazasi bolimlari uchun)
- Docker (deploy bolimi uchun)
