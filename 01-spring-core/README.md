# Spring Core

Spring Framework'ning eng asosiy qismi. Bu bo'limda Springning "ildiziga" tushamiz — IoC, DI va Bean qanday ishlashini tushunasiz.

## Mavzular

`01-what-is-spring.md` — Spring nima, nima uchun kerak, ekotizimga umumiy ko'rinish.

`02-ioc-container.md` — Inversion of Control nima, Container ichida nima bor, qanday ishlaydi.

`03-dependency-injection.md` — Constructor, Field va Setter Injection farqlari va qachon nima ishlatish kerak.

`04-bean-lifecycle.md` — Bean qanday yaratiladi, `@PostConstruct`, `@PreDestroy` va Scope'lar.

`05-application-context.md` — ApplicationContext tarkibi: Event tizimi, Environment, MessageSource.

## Qanday o'qish kerak?

Mavzularni ketma-ket o'qing. Har bir mavzu oldingisiga tayanadi — IoC'ni tushunmay turib DI'ni o'qish qiyin bo'ladi.
